What challenges did you encounter in training your agent?
The biggest hurdle was definitely the "Sparse Reward" problem. For the first few hundred episodes, the taxi just wandered around aimlessly, racking up -1 penalties every single second. Since the big +20 reward only happens at the very end when the passenger is dropped off, it took a long time for the agent to "backtrack" that success to the starting moves. I also had to be careful with the epsilon decay; if I stopped exploring too early, the taxi would just get stuck in a loop hitting the same walls because it hadn't found the passenger yet.

How did changing hyperparameters (learning rate, discount factor, epsilon) affect learning?
Changing the "knobs" on this algorithm changed the taxi's behavior completely.Epsilon ($\epsilon$): This was the most obvious one. When it was high (1.0), the agent was a total chaos-driver. As it decayed toward 0.01, you could actually see the "Success Rate" climb because the agent started trusting its own Q-table instead of just guessing.Learning Rate ($\alpha$): I found that keeping this low (0.1) worked best. I tried turning it up to see if it would learn faster, but the agent became too "reactive," changing its mind about a good route just because of one bad episode.Discount Factor ($\gamma$): Setting this to 0.9 was key. It made the agent value the "future" drop-off reward. Without a high gamma, the agent only cared about the -1 penalty for the next step and never really planned ahead to finish the job.

What are some real-world scenarios where Q-Learning or RL could be useful?
This is basically a pathfinding problem, so I could see this being used in automated warehouses (like Amazon) where robots have to find the best route to a shelf and back. It’s also used in dynamic pricing for apps like Uber, where the "agent" has to decide the best price based on demand to maximize profit. Another cool use is traffic light optimization, where the system learns the best timing to keep cars moving based on how many are sitting at an intersection.
